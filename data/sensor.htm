voltage divider With a simple voltage divider, you can also measure voltages greater than VCC (3.3V). If you want to measure voltage up to 36V, you can use an 11:1 (10k:1K) divider as in the example.

3.3V / ( R7 / ( R7 + R8 ) ) = voltage range
example:
3.3V / ( 1k / ( 1k + 10k ) ) = 36.6V
3.3V / ( 2k / ( 2k + 10k ) ) = 19.8V
3.3V / ( 2k / ( 2k + 1k ) ) = 4.95V
and the ratio:
3.3V / ( R7 / ( R7 + R8 ) ) / 4096 = ratio
3.3V / ( 1k / ( 1k + 10k ) ) / 4096 = 0.008862305 (36V)
3.3V / ( 2k / ( 2k + 10k ) ) / 4096 = 0,004833984 (19.8V)
3.3V / ( 2k / ( 2k + 1k ) ) / 4096 = 0,001208496 (4.95V)
